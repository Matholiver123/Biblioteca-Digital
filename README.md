# Biblioteca Digital

## Descrição
Sistema simples de gerenciamento de biblioteca, permitindo cadastrar livros e usuários, realizar empréstimos e devoluções. O projeto é construído em Java com foco em orientação a objetos.

## Funcionalidades
- Cadastrar livros (título, autor, ISBN, disponibilidade)
- Cadastrar usuários (nome, ID)
- Realizar empréstimos de livros para usuários
- Registrar devoluções de livros
- Exibir informações de livros, usuários e empréstimos
